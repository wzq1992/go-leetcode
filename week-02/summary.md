递归
-----------

* 函数自身调用自己

* 典型案例：阶乘
    
    ```go
    func factorial(n int) int {
        if n == 1 {
      	    return 1
        }
      
        return n * factorial(n - 1) 
    }
    ```
    
 * 关键点：子问题、边界、还原现场
 
 * 递归形式：指数型、排列型、组合型
 
 
 分治
 ---------
 把原问题划分成若干个同类子问题，分别解决后，合并
 
 * 子问题划分标准：不重不漏
